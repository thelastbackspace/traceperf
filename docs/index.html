TracePerf

Advanced console logging & performance tracking for Node.js

GitHub Get Started

Features

📊 Execution Flow Tracing

Track function calls and visualize the execution flow with ASCII art.

⏱️ Performance Monitoring

Identify slow functions and get suggestions for optimization.

🔀 Conditional Logging Modes

Different logging levels for development, staging, and production.

📚 Nested Logging

Group related logs with indentation for better readability.

🧹 Auto-Remove Debug Logs

Automatically filter out debug logs in production.

📤 Multiple Output Formats

Support for CLI and JSON output formats.

Getting Started

Installation

npm install traceperf

Basic Usage

const tracePerf = require('traceperf');

// Basic logging
tracePerf.info('This is an info message');
tracePerf.warn('This is a warning message');
tracePerf.error('This is an error message');
tracePerf.debug('This is a debug message');

// Logging with objects
tracePerf.info('User data:', { id: 1, name: 'John Doe' });

// Nested logging with groups
tracePerf.group('User Authentication');
tracePerf.info('Checking credentials');
tracePerf.warn('Invalid password attempt');
tracePerf.groupEnd();

Execution Flow Tracking

function fetchData() {
  processData();
}

function processData() {
  calculateResults();
}

function calculateResults() {
  return 'done';
}

// Track the execution of the fetchData function
const result = tracePerf.track(fetchData);
