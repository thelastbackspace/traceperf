Try it in the Console

The tracker is available in the console as traceperf. Try these examples:

// Track a simple operation
traceperf.track(() => {
    console.log('Hello from tracked function!');
}, { label: 'consoleTest' });

// Track an async operation
traceperf.track(async () => {
    await new Promise(resolve => setTimeout(resolve, 1000));
    return 'Done!';
}, { label: 'asyncTest' });
